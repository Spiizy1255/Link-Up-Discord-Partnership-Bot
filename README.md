## Main Commands

- `/setup` - Set up Link Up! for your server
- `/partner` - Preview and send your advert
- `/preview` - Preview without broadcasting
- `/premium` - View premium packages
- `/partners` - View network stats
- `/partnertracker` - View partnership leaderboard
- `/embedcolour` - Change advert embed colour
- `/banner` - Manage advert banners
- `/check` - Check bot permissions
- `/plan` - View your server plan
- `/help` - View all commands
